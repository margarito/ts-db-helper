Class QuerySelect<T>

class

QuerySelect

description

For design reasons this class should not be used directly. Use this class with Select function.

example
// select todos
Select(Todo).where({isDone: false}}).exec().subscribe((qr: QueryResult<Todo>) => {
     // do something with the result...
}, (err) => {
     // do something with the error...
});
